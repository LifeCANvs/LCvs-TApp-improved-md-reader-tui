# imd <file> [options]

## <file>

File to open.

IMD also supports pipping.

## Options

### -r, --raw

***NOT IMPLEMENTED***

Print data without less like feature.

### -d, --no-decoration

***NOT IMPLEMENTED***

No decoration I.E line numbers and file display

### -s, --no-syntax

***NOT IMPLEMENTED***

No syntax highlighting.

### -h, --help

Display this file.
